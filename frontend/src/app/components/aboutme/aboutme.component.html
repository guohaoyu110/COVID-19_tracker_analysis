About us

Hi, we are Haoyu Guo and Tianyi Xu, graduate students from University of Southern California in EE departement.

This is our first fullstack project trying to track the current status of the coronavirus outbreak all over the world.

This project started with simple thought of building data source of Coronovirus numbers after watching numbers on news baners. We built API related to data from John Hopkings University and frontend interface based on Angular framework.

You can see the source code of this project at GitHub: COVID-19_tracker_analysis

If you find error in reporting counts, feel free to pull request or post a issue.
